Yo Cat!

Yo cat, I heard you like open source software projects, so I recommend some open source software projects so you can write some open source software project code.

But seriously, Yo Cat! is a Github recommender based on the graph database Neo4j. It builds a graph of repositories and users and makes recommendations based on repositories and users you already like.
